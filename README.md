# 3DThingWithNode
to activate it you need to open a terminal in vscode
then you'll need to type "npm install" to download all the node_modules
the last step is typing in "npm run start" in the terminal then clicking the localhost it gives you while holding control on your keyboard.

The planet generation for saturn is in saturn.js
the skybox generation is in saturn.js
the moons generation is in saturn.js
the music generation is in song.jss
